Учебная среда для изучения мониторинга с использованием контейнеризации на базе Docker.

Проект включает:
- FastAPI приложение с генерацией метрик;
- Prometheus для сбора метрик;
- Grafana для визуализации
- Zabbix для мониторинга и визуализации
- Node Exporter для мониторинга хоста
- PostgreSQL для хранения данных Zabbix

Возможности:
- увеличение количества пользователей;
- создание заказов;
- рост очереди запросов;
- генерация ошибок;
- утечка памяти;
- нагрузка на CPU(процессор)

Запуск:
 1 docker compose up -d --build

 2 Перейти по адресам:
  - http://localhost:8080 - Приложение
  - http://localhost:9090 - Prometheus
  - http://localhost:3000 - Grafana
  - http://localhost:8081 - Zabbix
   
Эндпоинты:
http://localhost:8080/metrics - экспортируемые метрики приложения в формате Prometheus

http://localhost:8080/control - веб-интерфейс для управления нагрузкой и генерации событий

http://localhost:8080/health - отображает состояние приложения
